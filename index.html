Ongoing VLSI / Physical Design
Layout-Level Dummy Cell Watermarking for IP Protection

Automated VLSI IP-protection tool that embeds hidden owner identity as scattered dummy cells in GDSII layouts. Uses BASE64 encoding + SHA-256 cryptographic verification. Verified zero DRC violations and zero power impact on CMOS inverter layouts using Cadence Virtuoso + GPDK045 45nm. Includes a Streamlit web app for attack simulation.

Anti-Deepfake Audio Authentication System

Hardware-software system that embeds inaudible ultrasonic watermarks at the point of audio recording using ESP32 with INMP441 I2S microphone. Flask/Python backend handles cryptographic signing. A scikit-learn/librosa ML model detects AI-generated voices. React frontend for real-time verification.
